# Frontend API Abstraction

This document provides a reference implementation design for the API and Service layer abstractions in the frontend application. Using an API abstraction layer makes the UI components cleaner and isolates HTTP logic.

## 1. HTTP Client Configuration

Initialize an Axios (or Fetch) instance that automatically attaches the JWT token to requests.

```typescript
// src/api/client.ts
import axios from 'axios';

const API_BASE_URL = process.env.NEXT_PUBLIC_API_URL || 'http://localhost:8000';

export const apiClient = axios.create({
  baseURL: API_BASE_URL,
  headers: {
    'Content-Type': 'application/json',
  },
});

// Interceptor to attach token
apiClient.interceptors.request.use((config) => {
  const token = localStorage.getItem('access_token');
  if (token && config.headers) {
    config.headers.Authorization = `Bearer ${token}`;
  }
  return config;
});
```

## 2. Authentication Service

Abstract the `/auth` endpoints.

```typescript
// src/api/auth.ts
import { apiClient } from './client';
import { LoginCredentials, SignupData, AuthResponse, UserProfile } from '../types';

export const AuthService = {
  
  async login(credentials: LoginCredentials): Promise<AuthResponse> {
    // Note: FastAPI OAuth2PasswordRequestForm expects form-urlencoded
    const params = new URLSearchParams();
    params.append('username', credentials.email);
    params.append('password', credentials.password);

    const response = await apiClient.post('/auth/login', params, {
      headers: { 'Content-Type': 'application/x-www-form-urlencoded' }
    });
    return response.data;
  },

  async signup(data: SignupData): Promise<UserProfile> {
    const response = await apiClient.post('/auth/signup', data);
    return response.data;
  },

  async getMe(): Promise<UserProfile> {
    const response = await apiClient.get('/auth/me');
    return response.data;
  }
};
```

## 3. Voice Service

Abstract the token generation endpoint.

```typescript
// src/api/voice.ts
import { apiClient } from './client';

export interface TokenResponse {
  token: string;
  livekit_url: string;
  room_name: string;
  participant_identity: string;
}

export const VoiceService = {
  
  async getToken(roomName: string, participantName: string): Promise<TokenResponse> {
    const response = await apiClient.post('/api/token', {
      room_name: roomName,
      participant_name: participantName
      // participant_identity is auto-generated by the backend if omitted
    });
    return response.data;
  }
};
```

## 4. Hook Integration (Example)

Inside the `VoiceRoom` component, you use the service seamlessly:

```typescript
const handleConnect = async () => {
    try {
        setUiState('CONNECTING');
        // 1. Fetch token
        const data = await VoiceService.getToken('main-room', user.name);
        
        // 2. Connect LiveKit
        await liveKitRoom.connect(data.livekit_url, data.token);
        // ... handled by event listeners
    } catch (err) {
        setUiState('ERROR');
        setErrorMessage('Failed to connect to agent');
    }
}
```
